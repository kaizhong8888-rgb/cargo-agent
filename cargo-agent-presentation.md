# cargo-agent

## 智能 Rust 编程助手

> 具备自我进化能力的 AI 编程助手
> 专注于 Rust 开发生态

---

## 关于 cargo-agent

cargo-agent 是一个创新的 AI 编程助手，具备以下核心特点：

- **自我进化**：能够修改自身源代码，持续学习和改进
- **Rust 专精**：深度集成 Rust 工具链和生态系统
- **工具丰富**：内置 50+ 专业工具，覆盖开发全流程
- **记忆系统**：支持持久化记忆和知识管理
- **任务规划**：可将复杂需求分解为可追踪的子任务

---

| 类别 | 功能 |

## 代码分析能力

- **结构分析**：提取函数、结构体、枚举、特征定义
- **依赖分析**：分析 use 语句和外部 crate 依赖
- **复杂度评估**：函数长度、嵌套深度分析
- **模式检测**：unsafe、unwrap、todo 等代码模式
- **AST 级分析**：基于 syn 的完整语法树解析
- **代码质量评分**：0-100 分质量评估
- **重复代码检测**：发现相似代码块

---

## 智能重构与优化

- **安全重构**：添加/移除 derive、替换 unwrap、重命名
- **布尔简化**：简化复杂布尔表达式
- **现代化**：更新为现代 Rust 模式
- **性能优化**：检测性能瓶颈并提供优化建议
- **惯用法改进**：建议更符合 Rust 风格的写法
- **Clippy 集成**：运行 lint、分类结果、自动修复
- **基准测试**：micro-benchmark、实现对比、热点分析

## 工具生态集成

- 🔧 **开发工具**：cargo clippy、cargo test、cargo build
- 📊 **数据处理**：CSV/JSON 解析、过滤、排序、聚合

## 自我进化系统

- **源码修改**：安全地读取、写入、创建、删除文件
- **工具创建**：动态生成新工具并集成到系统
- **验证机制**：修改后自动运行 cargo check 确保正确性
- **记忆存储**：持久化知识和经验，支持标签和重要性分级
- **进化记录**：记录工具创建、代码变更、经验教训
- **自我反思**：分析过去行为，识别改进模式
- **技能管理**：创建、更新、删除领域专业知识包

## 快速开始

1. 安装 cargo-agent
2. 配置必要的环境变量（如 API 密钥）
3. 在 Rust 项目目录中运行
4. 使用自然语言描述需求
5. cargo-agent 将自动分析并执行任务

### 示例命令

- `分析 src/ 目录的代码质量`
- `为 lib.rs 生成单元测试`
- `创建一个新的 CLI 项目`
- `运行 clippy 并修复常见问题`

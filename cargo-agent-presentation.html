关于 cargo-agent

  • 记忆系统：支持持久化记忆和知识管理
  • 任务规划：可将复杂需求分解为可追踪的子任务

智能重构与优化

  • 安全重构：添加/移除 derive、替换 unwrap、重命名
  • 布尔简化：简化复杂布尔表达式
  • 现代化：更新为现代 Rust 模式
  • 性能优化：检测性能瓶颈并提供优化建议
  • 惯用法改进：建议更符合 Rust 风格的写法
  • Clippy 集成：运行 lint、分类结果、自动修复
  • 基准测试：micro-benchmark、实现对比、热点分析

快速开始

  1. 安装 cargo-agent
  2. 配置必要的环境变量（如 API 密钥）
  3. 在 Rust 项目目录中运行
  4. 使用自然语言描述需求
  5. cargo-agent 将自动分析并执行任务
